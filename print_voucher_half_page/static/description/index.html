Payment Receipt

(Receipt & Payment Voucher Print)

This application provides the users to print both receipt voucher and payment voucher. The user have option to print in full A4 page or Half A4 page. This application helps in all industry as it is part of daily operations.

Features:

  • Company detailed address with logo.
  • Invoice number and invoice date.
  • Total amount in numbers with currency symbol.
  • Authorization line With signature (Administrator, Manager, receiver).
  • Total amount in words with currency name {Eg:Hunderd Us Dollar}.
  • Voucher narration line.
  • Customer / Vendor name.
  • Contacts details attached In footer.
  • Option to print in Full A4 page or Half A4 Page.
